# FedLearn Architecture (Scaffold)

## Current State
This repository currently contains scaffold implementations for sectors 01-12 with core Rust crates and browser package skeletons.

## Workspace Modules
- `crates/fedlearn-core`: core primitives (determinism, adapter, continual, privacy, session, crypto)
- `crates/fedlearn-server`: aggregation server scaffold and round lifecycle
- `crates/fedlearn-wasm`: wasm bridge scaffold
- `packages/fedlearn-js`: TypeScript/browser package scaffold
